<h1 align="center">WAY</h1>

<p align="center">
  <strong>WAY</strong> es una solución móvil diseñada para optimizar y automatizar el proceso de recuperación de objetos perdidos en la universidad. Su objetivo es conectar de manera eficiente a quienes han extraviado pertenencias con quienes las han encontrado, evitando desplazamientos innecesarios a oficinas físicas.
</p>

<p align="center">
Esta aplicación permite a los usuarios reportar rápidamente objetos encontrados (como audífonos o mochilas) y ofrece una búsqueda inteligente para que los dueños puedan localizar sus pertenencias desde cualquier lugar, automatizando un proceso que antes era manual.
</p>
